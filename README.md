# QR Code Attendance Management System✅

A robust, real-time attendance tracking system built with Python. This project utilizes computer vision to scan QR codes, verify user identity, and log attendance automatically into a database/CSV file.

---

## 🌟 Key Features
* **Real-Time Scanning:** High-speed QR detection using OpenCV and PyZbar.
* **Instant Logging:** Automatically records Name, ID, Date, and Time.
* **Duplicate Prevention:** Logic to prevent multiple entries for the same user in a single session.
* **Automatic CSV Reports:** Generates daily attendance sheets compatible with Excel.
* **QR Generator Script:** Included utility to create custom QR codes for new users.

---

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Libraries:** * `OpenCV` (Image processing)
    * `PyZbar` (QR decoding)
    * `Pandas` (Data management)
    * `QRCode` (Code generation)
